# Aegis Workspace

This directory stores design notes, baseline snapshots, and specs used to keep non-trivial product and architecture work aligned over time.

- `BASELINE-GOVERNANCE.md` defines how baseline evidence is interpreted.
- `baseline/` stores dated snapshots of the project state.
- `specs/` stores approved or review-ready design/spec artifacts.
- `INDEX.md` lists documents created in this workspace.
